Estilos para textos Elementos inline

Además de los habituales elementos strong, em o u, con los que introduciríamos texto en negrita, cursiva y subrayado, también pueden utilizarse ins para destacar un texto como añadido o del para mostrarlo como eliminado.

Mediante el elemento address es posible dar formato a un texto para que aparezca como una dirección, por ejemplo:

CampusMVP
Avda. de Madrid 18
Vigo, Spain

El elemento blockquote, para introducir citas, puede contener un elemento footer para diferenciar las referencias, por ejemplo:

What an astonishing thing a book is.

Carl Sagan

Estilos para código, comandos y resultados

Para dar instrucciones a un usuario puede utilizarse el estilo kbd, indicándole qué combinaciones+de+teclas tiene que usar.

Para destacar los nombres de variable se puede usar el elemento var: la variable min contendrá el menú valor de la serie.

La salida resultante de un comando puede resaltarse con el elemento samp

También pueden introducirse porciones de texto con formato previo, habitualmente para mostrar código de ordenador, usando para ello el elemento pre. Por ejemplo:

normalize <- function(x) {
   x <- sweep(x, 2, apply(x, 2, min))
   x <- sweep(x, 2, apply(x, 2, max), "/")
   2*x - 1
}

Transformaciones y alineaciones Selectores de clase

La alineación del texto se determina asignando a los elementos el selector de clase adecuado: text-left, text-right, text-center, text-justify o text-nowrap, por ejemplo:

Texto sin ajuste y que no se dividirá al llegar al final del espacio disponible en el ancho de párrafo, continuando hasta incluso salir de la ventana del navegador.

Mediante selectores de clase también es posible aplicar transformaciones al texto contenido en un elemento. Para ello se utilizan clases como text-lowercase, text-uppercase y text-capitalize, para convertir a minúsculas, mayúsculas o poner en mayúscula la inicial de cada palabra, respectivamente.
